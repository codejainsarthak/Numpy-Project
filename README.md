## Projects Included

This repository will include multiple NumPy-based data analysis projects. Each project focuses on a different real-world scenario.

---

### 1. E-Commerce Sales Analyzer
- Data cleaning with missing values (NaN handling)
- Row-wise and column-wise sales analysis
- Adding new products and new sales days
- Finding best-selling product and best sales day
- Applying discounts and normalization

---

### 2. Student Performance Analyzer
- Handling missing marks in datasets
- Calculating total and average scores
- Identifying top-performing students
- Subject-wise performance analysis
- Dataset expansion and transformation

---

### 3. Weather Data Analysis System
- Cleaning incomplete weather records
- Temperature trend analysis
- Daily and weekly data comparison
- Identifying extreme weather conditions
- Statistical summaries using NumPy

---

### 4. Inventory Management Analysis
- Stock level analysis for products
- Identifying low-stock items
- Demand vs supply comparison
- Data updates with new inventory entries
- Category-wise aggregation

---

### 5. Financial Expense Tracker
- Monthly expense dataset analysis
- Category-wise spending breakdown
- Finding highest and lowest expenses
- Budget optimization insights
- Data normalization and comparison

---

### 6. Sports Performance Analyzer
- Player performance dataset analysis
- Score aggregation across matches
- Identifying best performers
- Match-wise trend analysis
- Ranking based on performance metrics
